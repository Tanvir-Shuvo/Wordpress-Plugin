A WordPress plugin that Custimize WordPress Login page and Admin Area.

== Description ==

The " New Custom " is a minimal setup custom plugin. The " New Custom " features give the ability to customize the WordPress admin login page and few admin areas according to the plugin's settings.

It helps to Customize login page styles and set some custom messages for the WordPress admin panel.

== Features ==

Admin Login page Features:

-- Customize the login logo URL from the admin panel on the plugin settings page.

-- Customize login page styles with Enable and Disable by just selecting the radio field. (if select enable radio box remember me checkbox will auto-fill up)

-- Customize login page message(under the logo) by putting any messages. 

Admin area Features:

-- Customize the admin footer message.

--  Customize toolbar items (Remove new post and comment links from the Toolbar) by clicking the checkbox. 

-- Customize the admin color scheme (Default color scheme for new users).


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click the " New Custom " menu in the admin panel and customize your own choice.
4. Use the Plugin's Settings to configure the plugin
